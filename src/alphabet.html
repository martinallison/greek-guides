The first thing to learn about Greek is its crazy alphabet. This guide’ll take you through what it looks like and how the letters (and certain letter combinations) are pronounced. Here’s what it looks like:

If you’re going on holiday for a week, you’ll have no problem if you can’t read Greek characters—but people will be impressed and confused if you can. Give it a go!

Pronunciation

The different characters look scary at first, but actually they mostly all produce sounds that exist in English, or are easy to approximate. The pronunciation of each letter or combination is also really consistent, so once you’ve learned which letters and combinations map to each sound, you can pronounce anything.

Here we’ve grouped them into similar types and sounds and mapped them to their English equivalents.

Vowels

All of these letters have sounds made in English. Easy.

{{ macros.pronunciation_defs(vowels) }}

Notice there are quite a few vowels that have the same sound. The number of ways to write ‘ee’ can even confuse native Greeks when it comes to spelling!

Also note that some combinations of letters don't make the sounds you'd expect, like ‘οι’, which you’d expect to be ‘oee’ (this doesn’t just apply to vowels, either). For now, it might help to learn each combination as if it was just one letter.

Throaty

These ones are slightly harder for English speakers since their sounds are made using the throat. Scottish and French people are sorted, though.

{% set throaty = [ ("γ", "**g**ame, but softer, a bit like the French ‘r’"), ("γκ, γγ", "**g**oat"), ("κ", "**c**ar"), ("χ", "**h**eart, but more raspy, like Scottish lo**ch**"), ] %}
{{ macros.pronunciation_defs(throaty) }}

The letters γ and χ have an extra rule. If γ comes before an ‘ee’ or an ‘e’ sound, then it becomes more like year and yes, respectively. If χ comes before those sounds then it's pronounced as if there’s a y in between it and the vowel, e.g. hyee and hye.

Tonguey

Another set of easy letters, these are all made with the tongue touching the top of your mouth.

That's it

As far as individual letters and sounds go, that's pretty much everything there is to the alphabet and sounds in Greek for practical use.

If you continue on with learning and speaking Greek, you'll discover that there's still a little more to learn about pronunciation depending on word stress, but we'll see that later on. For now, though, with practice, you'll be able to pronounce 95%+ of all Greek words!
