The most obvious thing about Greek is that it uses a different alphabet. Whereas English uses the Latin alphabet, Greek uses… well, the Greek one. Learning it is the gateway to reading and speaking Greek.

Overview

The Greek alphabet has only 24 letters. Here they are, upper- and lowercase, in order:

At first it might look confusing, but some of the letters are probably recognisable already.

Some look the same as their English (Latin) counterparts, some you might recognise if you do maths or science things, and some you’ve probably seen being misused on Greek-style yoghurt packaging.

Others look more mysterious: what on earth is a Ξ, or a Ψ? Well, we’ll demystify them and the rest of the letters of the alphabet in small groups.

Notice σίγμα has two lowercase letters: ⟨σ, ς⟩. The only difference is that ⟨ς⟩ is used in place of ⟨σ⟩ as the last letter in a word. E.g.: καλός instead of καλόσ, πως instead of πωσ, etc.

Many Greek first-person verb forms end in ⟨ω⟩, so whenever you want to say that you do (or don’t do) something, you’ll be using this letter (which is quite a lot of the time) and so knowing it unlocks the ability to use Greek verbs:

Done

Hopefully the strange letters now seem less mysterious: they mostly just represent sounds that already exist in English.

After some practice you’ll be able to recognise them, and disassociate them from letters they might look like in English. At some point when you look at a Greek word, it’ll just be Greek.

Until then, here’s a summary of things it’s useful to remember while practicing:

  • ⟨β⟩ is ‘v’, not ‘b’
  • ⟨ν⟩ is ‘n’, not ‘v’
  • ⟨ω⟩ is ‘o’, not ‘w’
  • ⟨ς⟩ is used instead of ⟨σ⟩ if it’s the last letter in a word
  • ⟨ω⟩ is a very common verb ending
  • ⟨η⟩ is ‘ee’, not ‘h’ or ‘n’
  • ⟨υ⟩ is ‘ee’, not ‘u’ or ‘y’
  • If you struggle with pronouncing ⟨γ⟩, use ‘g’
  • If you struggle with pronouncing ⟨χ⟩, use ‘h’

You might have noticed that there are no Greek letters for ‘b‘, ‘d’ or hard ‘g’ sounds (among others). In the next guide, we’ll see how Greek uses two-letter combinations to make those (and other) sounds.
