Β • β βήτα vita

Although it looks like a B, this is actually the Greek ‘v’

English equivalent: v

Ζ • ζ ζήτα zita

Same as English ‘z’ but ever so slightly softer

English equivalent: z

Ι • ι ιώτα iota

Like the ‘ee’ in meet, but shorter. A lot of the time when it’s unstressed and before another vowel, it sounds like the ‘y’ in yes

English equivalent: i

Λ • λ λάμδα lamda

Exactly like English ‘l’

English equivalent: l

Ο • ο όμικρον omikron

Like the ‘o’ in form but more open

English equivalent: o
