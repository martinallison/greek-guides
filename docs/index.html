Hello! Welcome to Greek guides: a bunch of guides to learning the Modern Greek language.

Modern Greek isn’t a language people typically learn (and there are only 13ish million speakers), so there’s not much out there if you want to give it a go without needing to know about linguistic stuff.

That’s why we, Maggie (a native Greek) and Martin (someone who's learning Greek), decided to put these more practical guides together.

We hope you like them! Give us feedback or ask us questions here.

Basics

Learn the alphabet, how to pronounce things and enough to be able to impress some Greeks if you go on holiday.

Note on romanisation: you’ll see the word romanisation a lot in the guides in this section.

This just shows how Greek text can be written using equivalent (kind of) letters from the Latin (English) alphabet. E.g., the Greek word μαμά is romanised as mama.

When introducing a Greek word, we’ll write its romanisation in brackets along with its meaning: μαμά (mama—‘mum’). After this section we’ll drop the romanisation part.
