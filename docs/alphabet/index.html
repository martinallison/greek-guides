The first thing to learn about Greek is its crazy alphabet. This guide’ll take you through what it looks like and how the letters (and certain letter combinations) are pronounced. Here’s what it looks like:

If you’re going on holiday for a week, you’ll survive without being able to read Greek characters (most things in tourist places are also signed in Latin characters) but people will impressed and confused if you can read them, so give it a go!

Pronunciation

It looks scary at first, but actually most Greek sounds exist in English or are easy to approximate. Greek pronunciation is also really really consistent, so once you’ve learned which letters and combinations map to each sound, you can read anything.

We’ve grouped similar letters together for easier digestion.
